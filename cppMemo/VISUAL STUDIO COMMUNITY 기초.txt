------------솔루션 탐색기---------------

 > 리소스 파일: 리소스 파일은 프로그램에 
포함된 이미지, 아이콘, 문자열 등과 같은
 리소스들을 관리하는 파일입니다. 
일반적으로 .rc 확장자

>소스 파일: 소스 파일은 C++ 소스 코드를
 포함하는 파일입니다. .cpp 확장자를 가지며, Visual Studio에서는 이 파일을 편집하여 C++ 코드를 작성하고 프로그램을 개발

>헤더 파일: 헤더 파일은 C++ 코드에서 사용되는 함수, 변수, 클래스 등의 선언을 포함하는 파일입니다. .h 확장자를 가지며,

 소스 파일에서 #include 지시문을 사용하여 헤더 파일을 포함할 수 있습니다. 

일반적으로 같은 프로젝트 내에서 여러 소스 파일에서 공통으로 사용되는 함수, 클래스 등의 선언을 헤더 파일에 모아놓고, 소스 파일에서 이를 include하여 사용

--------------프로젝트 빌드-------------
Build 메뉴에서 'Solution Build' 클릭

Visual Studio에서 프로젝트 빌드란, 소스 코드 파일을 컴파일하여 실행 가능한 프로그램 파일을
 생성하는 과정.
 프로젝트 빌드는 소스 코드 파일에 작성된 코드를
 기계어로 변환하여 프로그램을 실행할 수 있는 형
태로 만드는 과정이라고 할 수 있습니다.

Visual Studio에서는 빌드를 수행할 때, 
컴파일러가 C++ 소스 코드를 컴파일하고, 
오류나 경고가 있는지 검사하며,
 컴파일된 파일을 링크하여 실행 가능한 파일을 
생성합니다.
